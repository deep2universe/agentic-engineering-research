Bringt den Copilot Coding Agent direkt ins Terminal: interaktive REPL, Slash-Commands, Built-in-Tools, MCP-Integration, parallele Subagenten (`/fleet`), Custom Agents ueber `AGENTS.md` und Headless-Modus fuer CI/Scripting. Inkl. Feature-Uebersicht, Installations- und Setup-Guide, Senior-Developer-Patterns, Agentic Engineering mit MCP/Security, Workflow-Vergleich gegen andere Coding-Agents und einem druckreifen Cheat Sheet (Stand: April 2026).
